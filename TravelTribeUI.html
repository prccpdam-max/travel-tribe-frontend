Database Schema
PostgreSQL + PostGIS · 14 tables · Full relational model
👤 users
idUUID PKPrimary key (uuid_generate_v4)
firebase_uidVARCHARFirebase Auth UID (unique)
emailVARCHAREmail address (unique)
usernameVARCHARUnique handle
avatar_urlTEXTAWS S3 image URL
nationalityVARCHARUser's nationality
is_verifiedBOOLEANAccount verification status
📍 places
idUUID PKPrimary key
categoryVARCHARrestaurant, cafe, bar, hotel, attraction, hidden_gem
latitude/longitudeDECIMALGPS coordinates
locationGEOGRAPHYPostGIS point — enables radius search
avg_ratingDECIMALAuto-updated by trigger on review insert
is_hidden_gemBOOLEANMarks secret/undiscovered places
📝 posts + post_photos + post_tags + post_likes + post_saves + post_comments
captionTEXTPost description
like_countINTDenormalized counter (fast reads)
post_photosTABLEMultiple photos per post with sort order
post_tagsTABLENormalized tag storage (#surf, #cafe etc)
parent_comment_idUUID FKSelf-referential for nested replies
👥 groups + group_members + group_posts
slugVARCHARURL-friendly unique identifier
is_publicBOOLEANPublic vs private groups
roleVARCHARadmin, moderator, member
member_countINTDenormalized for fast listing
🤝 buddy_posts + buddy_requests
date_from/date_toDATETrip date range
activity_typeVARCHARexplore, cafe, hiking, nightlife, beach, food
languagesVARCHAR[]Array of language codes ['en','th','ja']
statusVARCHARpending, accepted, declined
💬 conversations + messages + conversation_participants
typeVARCHARdirect or group_chat
last_read_atTIMESTAMPTZPer-user read tracking (unread count)
is_deletedBOOLEANSoft delete for messages
🔔 notifications
typeVARCHARlike, comment, follow, message, buddy_request
entity_typeVARCHARpost, review, buddy_post, group
entity_idUUIDPolymorphic reference to the entity
is_readBOOLEANRead/unread state
